OmegaT related repositories

There are some tools I made for OmegaT. OmegaT is a computer-assisted translation (CAT) tool.

  • LinkBuilder plugin for OmegaT: This plugin converts URLs inside the glossary and notes panes in OmegaT into clickable URLs so that a user can open the URLs using a browser by clicking it.
  • WebFinder plugin for OmegaT: coming soon
